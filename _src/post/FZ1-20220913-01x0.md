# [Guide] Markdown Line Break Guide

```
Date-published: 2022-09-13-01x0
Tags: #Guide
```

---

## No Line Break

```
Hello(<-- No Spaces)
World
```  
Result:  
Hello
World

## Line Break (using Two Spaces)

```
Hello  (<-- Two Spaces)
World
```  

Result:  
Hello  
World  

---

## [OPTIONAL] Line Break (using br tags)

```
Hello <br>
World
```

Result:  
Hello <br>
World


## [NOT] a Line Break

```
Hello

World
```

Result:  
Hello

World
